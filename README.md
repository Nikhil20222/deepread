# DeepRead

DeepRead is an AI-powered research assistant that helps users understand long PDFs through AI-generated summaries, interactive knowledge graphs, and evidence-backed answers.

## Planned Features

- Upload PDF
- AI Summary
- Knowledge Graph
- Ask Questions
- Highlight Evidence
- Multi-document Support
